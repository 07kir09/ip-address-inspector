# IP Geo Checker

Простая консольная программа для проверки IP-адреса.

## Что умеет

- Удобное меню в терминале.
- Проверка IP вручную (можно несколько через запятую).
- Автоматическое определение вашего публичного IP.
- Показ данных: страна, регион, город, индекс, координаты, провайдер, ASN и т.д.
- Ссылка на Google Maps.
- Сохранение карты в папку `maps/`.
- Тесты на `pytest`.

## Установка

```bash
cd ip-adresses
python -m venv .venv
source .venv/bin/activate
pip install requests rich pytest
```

## Запуск

```bash
python main.py
```

В меню:

- `1` — ввести IP вручную
- `2` — определить IP автоматически
- `0` — выход

## Тесты

```bash
python -m pytest -q tests
```

## Важно

- Геолокация по IP примерная, это не точный адрес человека.
- Для работы нужен интернет.
